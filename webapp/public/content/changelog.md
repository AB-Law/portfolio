# Changelog

## [0.2.0] - 2026-03-15

### Added
- Added changelog content route and surfaced release history in-app.
- Added `/status` page with deployment and runtime metadata powered by new status utilities.
- Added command palette with searchable navigation shortcuts and suggestions.
- Added generated blog search indexing and image asset mapping for markdown-driven content.
- Added SEO metadata composition utilities and route-level metadata generation flow.
- Added Playwright end-to-end coverage for core journeys and Vitest coverage for filtering/content-search utilities.

### Changed
- Migrated blog source data loading from mixed content sources into a unified Markdown content pipeline.
- Refined project/blog browsing behaviors (sorting, filtering, and route navigation) for consistency.
- Updated layout, navbar, and footer to include operational and discovery paths.
- Expanded CI, build, and lint/test scripts with bundle-size and SEO validation hooks.
- Improved app-level route handling and metadata usage for cleaner content discoverability.

### Changed
- Updated navigation to include product maturity and operational routes for quick access.
